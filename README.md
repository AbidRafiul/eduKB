## About

* **en:** Educational web platform built to facilitate public understanding of Family Planning (Keluarga Berencana), including methods, myths, and reproductive health.
* **id:** Platform web edukasi yang dibangun untuk memfasilitasi pemahaman masyarakat tentang Keluarga Berencana (KB), termasuk metode, mitos, dan kesehatan reproduksi.

## ✨ Features

* **Interactive Recommendation:** Filter suitable contraceptive methods based on current conditions.
* **Educational Modules:** Detailed pages explaining the three phases of family planning and reproductive health.
* **Responsive Design:** Clean and mobile-friendly layout across all devices.

## 🚀 Getting Started

* Run `npm install` to install dependencies.
* Run `npm run dev` to start the local server at `http://localhost:3000`.
